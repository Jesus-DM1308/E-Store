
# Dev

1. Instalar dependencias ```npm install```
2. Clonar el .env.template y crear el .env
3. Ejecutar el comando ```npm run dev```
4. Generar Drizzle client```npx drizzle-kit generate``` y ```npx drizzle-kit migrate```
